# 📖 Leaderboard<a name="about-project"></a>

**Leaderboard** "A leaderboard" is a type of ranking or scoring system used in various contexts to display and highlight the performance or achievements of individuals, teams, or entities in a competitive or comparative setting. It's a simple website. I created it using ES6 and Webpack!

## 🛠 Built With <a name="built-with"></a>

1.HTML5 2. CSS3 3. JavaScript 4. Webpack 5. Linters

### Tech Stack <a name="tech-stack"></a>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Webpack</li
</ul>

### Key Features <a name="key-features"></a>

- **Arrow functions**
- **Async and wait js features**
- **API to get game Id**
- **Makes a list of names and scores**
- **Adds and removes a scores**
